# CodeArena Platform

A web-based platform connecting clients with developers through coding projects and quests.

## Features

- ** User Management**: Registration, authentication, role-based access
- ** Project Marketplace**: Clients post projects, developers apply
- ** Quest System**: Coding challenges with points and rankings
- ** Dashboards**: Role-specific views for clients and coders
- ** Leaderboard**: Points-based ranking system

## Tech Stack

### Backend
- Java 17
- Spring Boot 3.x
- Spring Security
- Spring JDBC
- Maven

### Frontend
- Thymeleaf
- HTML5/CSS3
- Vanilla JavaScript
- Responsive Design

### Database
- MySQL 8.0
